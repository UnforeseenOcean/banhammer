BanHammer API Overview

Here is the documentation and some examples that will help you understand the BanHammer API.

API Authentication

URL: All
Method: All
Success Response: 200 | 201
Error Response: 401
Example: 
curl -H "Authorization: Token {{ token }}" {{ schema }}://{{ request.get_host }}{% url 'api:target_list' %} Reveal your secret API key

List all targets

URL Params: None

Add a new target

URL: {% url 'api:target_list' %}
Method: POST
URL Params: action=[ban|allow]
Data Params: {"target":"string", "reason":"string", "target_type":"ip|domain|url|hash|user", "target_action":"ban|allow", "method="string"}
Success Response: 201
Error Response: 400
Examples: 
curl -d "target_action=ban" -d "target=evil.com" -d "reason=malware dropper" -d "target_type=domain" -d "method=paloaltonetworks_add_to_ebl" {{ schema }}://{{ request.get_host }}{% url 'api:target_list' %}

curl -d "target_action=ban" -d "target=astott" -d "reason=terminated" -d "target_type=user" -d "method=activedirectory_disable" {{ schema }}://{{ request.get_host }}{% url 'api:target_list' %}

Retrieve or delete a target

URL: {% url 'api:target_list' %}[target_id]
Method: GET | DELETE
URL Params: target_id=integer
Data Params: None
Success Response: 200
Error Response: 404 | 204
Examples: 
curl {{ schema }}://{{ request.get_host }}{% url 'api:target_detail' target_id=5 %}

curl -XDELETE {{ schema }}://{{ request.get_host }}{% url 'api:target_detail' target_id=5 %}
